# Prompt: Topic Outline Generation

Use this prompt to expand a topic into a structured outline before writing full documentation.

```text
Create a comprehensive outline for the topic: <TOPIC>.

Context:
- Audience: experienced programmer growing practical systems and embedded Linux expertise.
- Repository style: concise Markdown documentation with examples.
- Do not write full explanations yet.
- Group items from beginner to advanced.
- Mark items especially relevant to embedded Linux.
- Include prerequisite topics and related topics.
- Prefer concrete mechanisms, commands, APIs, and failure modes over vague concepts.

Output:
1. Short scope statement.
2. Prerequisites.
3. Beginner outline.
4. Intermediate outline.
5. Advanced outline.
6. Embedded Linux relevance.
7. Suggested example files.
8. Related documentation pages.
```
