# Signal Processing Examples

Use this directory for scripts that support the signal-processing documentation.

Preferred workflow:

1. Put the source script under a subtopic directory, for example `signals-and-systems/`.
2. Generate SVG figures into `docs/assets/signal-processing/<subtopic>/`.
3. Link the generated figure from Markdown and reference the script near the image.

Keep scripts deterministic. If a random signal is required, set the random seed in the script and mention it in the caption or nearby text.
