## Scope

Signals And Systems owns the continuous-time and discrete-time math: signal models, system models, convolution, state-space descriptions, transform methods, stability, sampling foundations, and nonlinear-system approximations.

Keep implementation details here only when they clarify the theory. Move production-style algorithms and reusable code patterns to the programming page or to the DSP branch.

## Learning Order

1. Signal vocabulary and signal classes
2. Continuous-time LTI systems
3. State-space models
4. Fourier, Laplace, DFT, DTFT, and z-transform foundations
5. Frequency-domain system analysis
6. Discrete-time systems and difference equations
7. Nonlinear systems and numerical methods
8. Sampling, reconstruction, and continuous-to-discrete simulation
