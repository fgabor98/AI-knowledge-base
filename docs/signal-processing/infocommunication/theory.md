### 1. Communication System Model

- baseband vs carrier transmission
- bandwidth
- attenuation
- noise
- signal-to-noise ratio

- codewords
- average codeword length

### 3. Error-Control Coding

- Hamming distance

- amplitude modulation
- DSB, DSB-SC, and SSB
- envelope detection
- coherent demodulation

- image-frequency rejection

- PCM
- PAM
- symbol rate and bit rate

- decision thresholds

- ASK

- FSK
- QAM
- constellation diagrams
